# 管理系统项目

## 项目简介

这是一个基于 Vue 3 + Pinia + Tailwind CSS 构建的现代化管理系统，提供了仪表盘、数据统计、权限管理等核心功能，支持深色/浅色主题切换，具有良好的用户体验和视觉效果。

## 技术栈

### 前端框架与库
- **Vue 3**：采用 Composition API 构建组件
- **Pinia**：轻量级状态管理库，替代 Vuex
- **Tailwind CSS**：实用优先的 CSS 框架，用于快速构建响应式界面
- **Font Awesome**：图标库，用于界面图标展示
- **ECharts**：数据可视化图表库，用于数据统计展示
- **Vue Router**：路由管理

### 开发工具
- **Vite**：现代化前端构建工具，提供快速的开发体验
- **ESLint**：代码质量检查工具
- **Prettier**：代码格式化工具

## 项目架构

### 目录结构
```
src/
├── assets/          # 静态资源
├── components/      # 组件
│   ├── Layout/      # 布局组件（Sidebar、Topbar）
│   └── UI/          # UI 组件（BaseCard、BaseButton 等）
├── data/            # 模拟数据
├── views/           # 页面视图
├── router/          # 路由配置
├── stores/          # Pinia 状态管理
├── utils/           # 工具函数
└── App.vue          # 根组件
```

### 核心组件

#### 布局组件
- **Sidebar.vue**：侧边导航栏，包含系统功能菜单
- **Topbar.vue**：顶部导航栏，包含搜索、通知和主题切换

#### UI 组件
- **BaseCard.vue**：卡片组件，用于内容展示
- **BaseButton.vue**：按钮组件，支持多种样式和尺寸
- **BaseLineChart.vue**：折线图/柱状图组件，用于数据可视化
- **BasePieChart.vue**：饼图组件，用于数据分布展示
- **BaseTag.vue**：标签组件，用于状态展示
- **BaseProgress.vue**：进度条组件，用于系统状态展示
- **BaseModal.vue**：模态框组件，用于表单和详情展示

### 状态管理

使用 Pinia 进行状态管理，主要包括：
- **theme.js**：主题状态管理，控制深色/浅色模式切换
- **user.js**：用户状态管理，存储用户信息和权限

### 路由配置

系统包含以下主要路由：
- **/**：仪表盘页面
- **/dashboard**：仪表盘页面
- **/data-statistics**：数据统计页面
- **/permission-management**：权限管理页面
- **/order-management**：订单管理页面

## 实现功能

### 仪表盘（Dashboard）
- **统计卡片**：展示总访问量、用户总数、订单总数、总收入等关键指标
- **最近订单**：表格形式展示最近的订单信息，包含订单号、客户、日期、金额、状态等
- **用户活动统计**：展示新用户注册、新订单、访问量、消息通知等活动数据
- **系统状态**：展示 CPU 使用率、内存使用率、磁盘使用率、网络带宽等系统指标

### 数据统计（DataStatistics）
- **销售趋势**：折线图展示销售额的变化趋势
- **产品分类分布**：饼图展示不同产品分类的销售分布
- **月度销售数据**：表格形式展示月度销售额、订单数、客单价和环比数据

### 权限管理（PermissionManagement）
- **角色列表**：表格形式展示系统角色，包含角色名称、描述、用户数、状态等
- **权限分配**：树形结构展示系统权限，支持为角色分配不同的权限

### 订单管理（OrderManagement）
- **订单列表**：表格形式展示所有订单，支持筛选和搜索
- **订单详情**：展示订单的详细信息，包括商品、客户、支付等信息

### 主题切换
- 支持深色/浅色模式切换，适应不同的使用环境和个人偏好
- 主题设置会被保存，下次访问时保持上次的选择

## 模拟数据

项目使用模拟数据进行展示，主要存储在 `src/data/` 目录：
- **statistics.js**：包含销售趋势、产品分类等数据
- **orders.js**：包含订单相关数据
- **permissions.js**：包含权限相关数据

## 项目特点

1. **现代化技术栈**：使用 Vue 3、Pinia、Tailwind CSS 等最新技术
2. **响应式设计**：适配不同屏幕尺寸，提供良好的移动端体验
3. **组件化开发**：封装了一系列可复用的 UI 组件，提高开发效率
4. **数据可视化**：使用 ECharts 实现丰富的数据图表展示
5. **主题切换**：支持深色/浅色模式，提升用户体验
6. **代码质量**：使用 ESLint 和 Prettier 保证代码质量和一致性
